# Contributing to GitQuest

## Branch Naming
Use one of these patterns:
- `feature/<your-name>`
- `fix/<your-name>`
- `chore/<your-name>`

Examples:
- `feature/alex`
- `fix/sana-conflict`
- `chore/rahul-readme`

## Commit Message Format
Use:
`type: short description`

Examples:
- `feat: add student intro file`
- `fix: resolve merge conflict`
- `chore: update changelog`

## Steps to Contribute
1. Create a branch
2. Make changes
3. Commit
4. Push
5. Create a Pull Request
6. Wait for review and merge

## Pro Tip
Pull latest `main` before you start:
```bash
git checkout main
git pull origin main
```
